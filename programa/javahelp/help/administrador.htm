Colegio – Menú de Ayuda


Menú de administrador

Como administrador puede editar toda la información y datos personales del resto de usuarios de la aplicación y de usted mismo.

También puede añadir usuarios nuevos, y borrar usuarios existentes.

Desde el menú de cursos podrá crear, modificar y borrar cursos [NYI]

También puede realizar backups completos del estado actual de la base de datos y restaurar la base de datos actual a estados anteriores utilizando estos ficheros.

Usuarios

El menú usuarios muestra una tabla con la información de todos los usuarios registrados en la aplicación.

Puede editar y modificar los campos de los usuarios, sólo tiene que hacer doble click sobre el campo a modificar. Al terminar de moficar el campo, los cambios se almacenan automaticamente en la base de datos externa.

Si hubiera algún problema con las modificaciones realizadas, se le advertirá mediante una ventana popup.

Al botón derecho se despliega un menú emergente con tres opciones: añadir nuevo usuario, borrar usuario seleccionado y cambiar contraseña.

  • Cada vez que seleccione añadir usuario, se añadirá una nueva línea a la tabla, y podrá modificar los campos que desee. En el momento que la línea contenga datos válidos, estos se almacenarán de forma automática en la base de datos.

Las líneas en blanco añadidas pero no rellenadas o rellenadas con datos no válidos se ignoran y no se añaden a la base de datos

  • Respecto al borrado de usuarios, tenga en cuenta que no se permite borrar al usuario que está actualmente conectado al sistema.

  • Puede modificar la contraseña de cualquier usuario de la aplicación, incluida la suya.

Recuerde la contraseña introducida ya que, por razones de seguridad, esta no se muestra ni almacena nunca en la aplicación.

Las contraseñas sólo se almacenan en la base de datos externa, y siempre encriptadas, por lo que si no la recuerda no podrá recuperarla y tendrá que contactar con otro administrador.

Cursos

[NYI]

Hacer backup

Desde ese menú podrá realizar una copia de seguridad de la base de datos actualmente en uso, con todas sus tablas y los datos contenidos en las mismas.

Debe seleccionar la ruta y el nombre del fichero donde desea guardar el backup (asegúrese de que tiene permisos de escritura en la ruta elegida).

No es necesario que indique la extensión del fichero (esta será añadida automaticamente).

Restaurar backup

Permite restaurar la base de datos a un estado anterior, con todas sus tablas y los datos contenidos en las mismas.

Para ello, debe seleccionar un fichero de backup válido (extensión .sql) que haya sido creado previamente con la aplicación.

[AVISO] Para poder concluir el proceso de restauración, es necesario reiniciar la aplicación: esta se cerrará de forma automática y deberá volver a arrancarla manualmente.

La próxima vez que arranque la aplicación, esta se habrá actualizado con los datos del backup seleccionado.
